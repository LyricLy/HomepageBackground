# Homepage Background
A Town of Salem 2 mod to replace the background on your home screen with a custom image. Can also be used to hide the news panel.

## Installing
This mod requires [Salem Mod Loader](https://github.com/Curtbot9000/SalemModLoader), but is not available on its mod browser yet.
After installing SML, you can [download the latest release of Homepage Background](https://github.com/LyricLy/HomepageBackground/releases/latest) and copy the .dll file to the `Town of Salem 2/SalemModLoader/Mods/` directory.

## Configuring
Place your background image in `Town of Salem 2/SalemModLoader/ModFolders/HomepageBackground/background.png`. In-game configuration in the Salem Mod Loader menu can be used to enable hiding the news panel.
